Welcome to pycompwa’s documentation!¶

pycompwa is a collection of python modules which extend the Common Partial-Wave-Analysis Framework (ComPWA). These docs are meant to provide information about the installation, howtos to get things running, information on the design, as well as documentation of the ongoing development!
